Barre de subgoal s'actualisant automatiquement à chaque sub ou subgift
Possibilité de paramétrer jusqu'à 10 subogals
Lorsqu'un goal est atteint la barre se fige quelques secondes avant d'afficher automatiquement le subgoal suivant

Se rendre et lié son compte twitch sur https://streamelements.com/dashboard/overlays
Cliquer sur NOUVELLE OVERLAY
Régler la résolution comme souhaité
Renommer l'overlay et cliquer sur ADD WIDGET
Sélectionner STATIC/CUSTOM puis Custom widget.
Réduire Fields
Se rendre dans Position, size and style et régler les paramètres comme ci-dessous
> Width : 908 PX
> Height : 68 PX
Cliquer sur CENTER WIDGET pour une meilleure visibilité de celui pendant la configuration
Réduire Position, size and style et développer Settings, puis cliquer sur OPEN EDITOR
Remplacer le code des onglets HTML CSS JS FIELDS et DATA, petites précisions : 
> FIELDS correspond au fichier .json
> Pour gagner du temps cliquer sur chaque code et faire "Ctrl A" et "Suppr" pour tout sélectionner et supprimer plus rapidement
> Pour rentrer le nouveau code il suffit simplement de glisser le fichier .html, .css, etc à coter du "1" dans l'onglet souhaité
Cliquer sur DONE
Pour paramétrer la barre à sa convenance développer Fields
Pour les images de la barre de chargement et de fond, utiliser des liens web où apparait le type de fichier (ex: .png, .jpg, .gif)
Pour uploader une image facilement et gratuitement utiliser le site imgur.com
Pour récupérer le lien correctement, faire une clic droit "copier l'adresse de l'image"
Le champs remplissage permet de régler la taille de l'image dans les barres, si celle ci est trop grande
Cliquer sur SAVE en haut à gauche une fois la configuration terminée
Pour récupérer le lien de la source cliquer sur le petit symbole à gauche de PREVIEW

Il est également possible d'effectuer des tests, de la même manière qu'avec streamlabs, en jouant avec la cloche EMULATE
